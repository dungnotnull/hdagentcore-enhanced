## How to Contribute

We welcome contributions! Please see [CONTRIBUTING.md](CONTRIBUTING.md) for guidelines.

### Ways to Contribute

1. **Code**: Submit pull requests with new features or bug fixes
2. **Documentation**: Improve docs, fix typos, add examples
3. **Tests**: Write tests for uncovered code
4. **Issues**: Report bugs, request features, ask questions
5. **Reviews**: Review pull requests and provide feedback
6. **Sharing**: Share your agent patterns and use cases
